WAI_ARIA

Web Accessbility Initiative's Accessible Rich Internet Applications

스크린리더가 브라우저를 읽을 때 각 요소가 어떤 역할을 하는지 알 수 있도록 한다.

role 속성으로 태그 역할을 지정하고, aria-label로 이름표를 붙여주면 스크린 리더가 읽을 수 있다.

role

role은 태그의 역할을 알려주는 속성(attribute)이다

MDN : role

aria-label

aria-label은 태그가 가지고 있는 의미를 적어주는 이름표 역할을 한다.

  1. role처럼 특정 값이 정해져 있는 게 아니라, 브라우저가 스크린 리더 사용자에게 전달 할 내용이 들어간다.
  2. 시멘틱 태그이거나, role 속성 값이 상호작용 하는 값일 때 aira-label을 사용할 수 있다.
  3. 시맨틱 태그가 아닐 경우, role 속성이 없으면 aria-label 사용 불가.
  4. 시멘틱 태그에는 가능한 role을 사용하지 않는다.
  5. 시멘틱 태그가 의미하는 것을 aria-label로 나타낼 필요없다.
